①  プロジェクト名： Kaggleデータを用いた配送コースの利益向上

ファイル名：Logistics_Analytics.ipynb

背景・目的： 
物流業界の2024年問題や燃料費高騰に対し、データ分析によるアプローチで「短距離ルートの逆ザヤ（赤字）」という課題を特定。単なるデータ集計に留まらず、現場のコース改善の道標の発見から、経営に直結する運賃改定計画のシミュレーションまでを構築しました。

使用技術・ライブラリ：
Python, Pandas, NumPy, Matplotlib, Seaborn

成果物の見どころ：
配送データと顧客データを結合（JOIN）し、ルートごとの収益性をマトリクスと散布図で可視化する前処理・分析ロジック・運賃改定までの一連のシナリオを作成。


②  プロジェクト名： 【Kaggleコンペ】中古車の価格予測(6/24～現在作成中)

ファイル名： Usedcar_price_prediction.ipynb

背景・目的：
実務におけるデータ分析において高頻度で発生する「データの欠損・記入漏れ・表記揺れ」といった課題を想定。そのままでは活用できない不完全なデータに対し、機械学習を用いて他の説明変数から高精度に予測・補完し、ビジネスで使えるデータへと修復（データマネジメント）技術の実証を目的として構築しています。

使用技術・ライブラリ：
Python, Pandas, NumPy, Matplotlib, Seaborn, LightGBM

成果物の見どころ：
単にモデルを動かすだけでなく、データ構造の理解（EDA）、外れ値処理、欠損値の高度な補完（ imputation ）、そしてLightGBMを用いたハイパーパラメータチューニングまでの一連の機械学習パイプラインを自力で実装している点。
