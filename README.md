①  プロジェクト名： Kaggleデータを用いた配送コースの利益向上

ファイル名：Logistics_Analytics.ipynb

背景・目的： 物流業界の2024年問題や燃料費高騰に対し、データ分析によるアプローチで「短距離ルートの逆ザヤ（赤字）」という課題を特定。単なるデータ集計に留まらず、現場のコース改善の道標の発見から、経営に直結する運賃改定計画のシミュレーションまでを構築しました。

使用技術・ライブラリ：

Python, Pandas, NumPy, Matplotlib, Seaborn

成果物の見どころ：配送データと顧客データを結合（JOIN）し、ルートごとの収益性をマトリクスと散布図で可視化する前処理・分析ロジック・運賃改定までの一連のシナリオを作成。
